User Instructions

How to Use the Locations Page

The Locations page helps you find the shelf location of products that already have a documented storage spot. If a product is new or doesn’t have a location assigned, you’ll first need to create a location for it and then update its assigned spot using this page. For help with that process, see the "Adding Locations" section below.

Finding a Product’s Location

  1. Identify the Part Number
    • Look it up in Odoo
    • Or check the product’s physical label
  2. Search for the Part
    • Click the open search field on the Locations page
    • Enter the part number
    • Note: The search is not case-sensitive. Just make sure the number matches what's shown in Odoo or on the label
    • Click Search
  3. Read the Results
    • Search results will appear just below the Search button in the format: #-A-# (for example: 3-A-5)
    • First Number = Shelf row number (e.g., Row 3)
    • Letter = Side of the shelf (A = front, B = back)
    • Last Number = Slot number where the item is stored
  4. Locate and Verify
    • Go to the indicated shelf and slot. Inside the slot, you’ll find:
    • The slot number
    • The name of the product
    • Double-check that the product in the slot matches what’s listed.

How to make updates on the Updates page

Adding a new part and location

The following will walk you through how to add a new part to the Finder.

  1. Enter the Part Number
    • In the "Enter Part Number" field, type the part number exactly as it appears in Odoo
    • Make sure it matches Odoo formatting to ensure proper tracking
  2. Assign a Location
    • Enter the shelf location where the part will be stored using the format: Shelf-Section-Slot (Example: 1-A-3)
    • Shelf = Row number
    • Section = A (front) or B (back)
    • Slot = Specific position on the shelf
  3. Add the Part
    • Click the Add button
  4. Verify the Information
    • A popup will appear asking you to confirm the details you entered
    • Carefully check for accuracy, then click Submit
    • Once submitted, the new part and its assigned location will be added to the system database.

Changing a products location

If you need to update the location of a product, please keep the following in mind: Only one location is updated at a time. If the product exists in multiple locations, only the specific instance you select will be updated. To update all locations for a product, you will need to make changes one location at a time.

  1. Enter the Part Number
    • In the "Enter Part Number" field, type the part number exactly as it appears in Odoo
  2. Enter the Current Location
    • Input the product’s current location (e.g., 1-B-23) in the "Current Location" field
  3. Enter the New Location
    • In the "Enter New Location" field, type the new shelf location where the part should be moved (e.g., 2-A-10)
  4. Apply the Change
    • Click the Change button
  5. Review and Confirm
    • A popup will appear asking you to verify the information
    • Review the details for accuracy, then click Confirm

Removeing a part and location

When removing a product from a specific location, please note: Only the selected location is removed. If the part exists in multiple locations, you’ll need to remove each one individually. For example: If part 3015 is stored in five different slots, you must delete each slot entry one at a time.

  1. Enter the Part Number
    • In the "Enter Part Number" field, type the part number exactly as it appears in Odoo
    • Make sure to verify the number in Odoo before continuing
  2. Enter the Product’s Location
    • In the "Enter Product Location" field, input the current location of the part (e.g., 2-A-7)
  3. Delete the Entry
    • Click the Delete button
  4. Review and Confirm
    • A popup will appear asking you to confirm the removal
    • Double-check the details, then click Confirm to permanently remove the part from that location
    • The selected product and location entry has now been successfully deleted from the system
